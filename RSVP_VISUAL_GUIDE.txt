┌──────────────────────────────────────────────────────────────────────────────┐
│                        SECURITY MODEL                                         │
└──────────────────────────────────────────────────────────────────────────────┘

    Token Generation:
    ┌────────────────────────────────────────────────────────┐
    │  secrets.token_urlsafe(32)                             │
    │  → Generates 32-byte cryptographically secure token    │
    │  → URL-safe base64 encoding                            │
    │  → Example: "abc123XYZ-_def456..."                     │
    └────────────────────────────────────────────────────────┘

    Token Validation:
    ┌────────────────────────────────────────────────────────┐
    │  1. Check token exists in database                     │
    │  2. Verify invitation is active                        │
    │  3. Grant access to RSVP form                          │
    │  4. No other authentication needed                     │
    └────────────────────────────────────────────────────────┘
